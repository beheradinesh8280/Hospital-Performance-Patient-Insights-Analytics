🏥 Hospital Performance & Patient Insights Analytics

A Power BI dashboard template designed to analyze hospital operations, patient behavior, and financial performance. This project converts raw healthcare data into actionable insights for better decision-making.

📌 Overview

This project focuses on analyzing key hospital metrics such as patient flow, revenue, and operational efficiency. It helps identify trends, optimize resources, and improve overall hospital performance.

🚀 Features
Interactive dashboards with filters and slicers
Patient admission and discharge analysis
Readmission rate tracking
Emergency vs scheduled visit insights
Doctor performance ranking
Department-wise revenue analysis
Time-based trends (daily, monthly, yearly)
📊 Key Metrics
Total Patients
Total Revenue
Readmission Rate
Average Length of Stay
Emergency Visits
Department-wise Performance
Doctor Rankings
🛠️ Tools & Technologies
Power BI
DAX (Data Analysis Expressions)
Power Query
Excel / CSV
