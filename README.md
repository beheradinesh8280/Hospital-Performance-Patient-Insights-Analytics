# 🏥 Hospital Performance & Patient Insights Analytics (Power BI)

---

## 📌 Project Overview

The **Hospital Performance & Patient Insights Dashboard** is a comprehensive analytics solution built using **Power BI** to monitor, evaluate, and improve hospital operations.

This project transforms raw healthcare data into **actionable insights** that support better decision-making. It provides a centralized view of **patient activity, hospital performance, and financial outcomes**.

By analyzing trends such as **patient admissions, treatment efficiency, and revenue generation**, the dashboard helps stakeholders identify inefficiencies and optimize operations.

This solution is ideal for:
- Hospital Administrators  
- Healthcare Analysts  
- Decision Makers  

---

## 🎯 Objectives

The main goal of this project is to build a **data-driven hospital analytics system**.

- Analyze **patient flow** (admissions, discharges, readmissions)
- Track **hospital revenue and financial growth**
- Measure **doctor and department efficiency**
- Identify patterns in **emergency vs scheduled visits**
- Provide **actionable insights** for operational improvement

---

## 🛠️ Tools & Technologies Used

- **Power BI** – Dashboard & Visualization  
- **Power Query** – Data Cleaning & Transformation  
- **DAX** – KPI Calculations & Measures  
- **Excel / CSV** – Data Source  
- **Data Visualization Techniques**

---

## 📊 Dataset Information

The dataset represents **hospital operational data**, including:

- Patient ID, Admission & Discharge Dates  
- Admission Type (Emergency / Scheduled)  
- Department & Doctor Details  
- Treatment Costs & Revenue  
- Patient Demographics  
- Length of Stay & Diagnosis  

These attributes help analyze **efficiency, performance, and patient outcomes**.

---

## 📈 Key Performance Indicators (KPIs)

- **Total Patients**
- **Total Revenue**
- **Readmission Rate**
- **Average Length of Stay**
- **Emergency Visits**
- **Scheduled Visits**
- **Department-wise Revenue**
- **Doctor Performance Ranking**

---

## 🚀 Dashboard Features

### 1. KPI Summary Section
Provides a quick overview of key hospital metrics such as **patients, revenue, and efficiency indicators**.

---

### 2. Patient Admission Analysis

Analyzes patient admission trends over time to identify **peak demand periods**.

---

### 3. Department Performance Analysis

Evaluates departments based on:
- Patient volume  
- Revenue contribution  

Helps identify **top-performing and underperforming departments**.
---

### 4. Doctor Performance Analysis

Ranks doctors based on:
- Number of patients treated  
- Revenue generated  

---

### 5. Interactive Filters

Users can dynamically filter data by:
- Department  
- Doctor  
- Admission Type  
- Time Period  

---

## 🔍 Key Insights Derived

- High-performing departments contribute significantly to revenue  
- Readmission patterns highlight potential treatment gaps  
- Emergency visits fluctuate based on time trends  
- Doctor performance varies across departments  

These insights help improve **hospital efficiency and patient care**.
